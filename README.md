# dotfiles

## Directory and renamed files for instalation:

**Vim**

- Rename and move the file to the user directory:

```
$ mv vimrc ~/.vimrc
```

**Git config**

- Rename and move the file to the user directory:

```
$ mv gitconfig ~/.gitconfig
```

**Zsh**

- Rename and move the file to the user directory:

```
$ mv zshrc ~/.zshrc
```
